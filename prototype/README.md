# Venture OS Prototype

Static product prototype for the Venture OS workspace.

The visual direction blends:

- Figma-style dark community/resource browser, chips, cards, workspace rail, and create modal.
- Cursor-style command center, agent queue, diff/review panel, and approval-oriented workflow.

## Open

Open [`index.html`](./index.html) in a browser.

## Intent

This is not a marketing site. It is the first-screen operating surface for a
founder or venture team running fundraising, growth, visibility, investor
relations, partnerships, and collateral workflows.

## Porting Path

1. Keep this static prototype as the visual baseline.
2. Move to `Next.js` or the selected product shell when Venture OS becomes an app.
3. Replace static data with the Venture OS data model:
   - clients
   - opportunities
   - source signals
   - agents
   - deliverables
   - approvals
   - briefs
