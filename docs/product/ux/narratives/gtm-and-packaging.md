# Venture OS GTM and Packaging

## Product Names

Candidate product names:

| Name | Position |
| ---- | -------- |
| Venture OS | System/product category |
| Capital Desk | Fundraising-led managed service |
| Venture Desk | Broader managed service for fundraising, growth, visibility, partnerships |
| Founder Command Center | App/dashboard surface |
| Africa Capital Intelligence | Data/intelligence module |

Recommended external packaging:

> **Venture Desk by GTCX**  
> A managed AI-powered fundraising, growth, and visibility desk for African and
> Global South ventures.

## Offers

| Offer | Buyer | What They Get |
| ----- | ----- | ------------- |
| Founder Sprint | Single startup | 30-day pipeline build, positioning, outreach, collateral, weekly brief |
| Capital Desk | Startup/studio/fund | Ongoing fundraising and investor relations workflow |
| Growth Desk | Startup/corporate program | Revenue, pilots, partnerships, channel pipeline |
| Visibility Desk | Founder/executive/team | LinkedIn, press, events, awards, newsletter, thought leadership |
| Cohort Desk | Accelerator/incubator | Shared opportunity database and weekly founder support |
| Portfolio Desk | Fund/foundation | Portfolio company support and opportunity routing |
| Intelligence License | Hub/fund/corporate | Access to Africa/Global South opportunity/source database |

## Pricing Logic

Use service-led pricing first. SaaS pricing comes after repeated workflows are
stable.

| Package | Indicative Price Logic |
| ------- | ---------------------- |
| Founder Sprint | Fixed fee for 30 days |
| Capital/Growth/Visibility Desk | Monthly retainer by client complexity |
| Cohort Desk | Program fee by cohort size |
| Portfolio Desk | Annual or quarterly platform/support fee |
| Intelligence License | Subscription by seats, sources, and coverage |
| Enterprise/DFI Program | Custom annual contract |

## Early Customer ICPs

| ICP | Strong Fit Signal |
| --- | ----------------- |
| African startup raising seed/Series A | Founder has product, traction, and investor/partner confusion |
| Venture studio | Multiple portfolio companies need the same fundraising/growth office |
| Accelerator/incubator | Wants better post-program funding and visibility outcomes |
| Fund/foundation | Needs portfolio support without hiring a large internal team |
| Ecosystem hub | Wants country/sector opportunity intelligence and founder services |
| Corporate/DFI program | Needs credible venture pipeline and program engagement |

## Sales Motion

1. Start with warm clients.
2. Offer a 30-day Venture Diagnostic.
3. Deliver immediate pipeline and collateral value.
4. Convert to monthly Desk retainer.
5. Aggregate anonymized workflow learnings into app requirements.
6. Sell cohort/portfolio packages once repeatable.

## Core Sales Narrative

African founders do not only need capital. They need a venture operating system:

- Find the right opportunities.
- Know what to ignore.
- Produce investor-grade materials.
- Build visibility.
- Convert relationships into pilots, revenue, and capital.
- Stay organized while running the company.

## Proof Strategy

Use GTCX portfolio as proof:

- Show before/after pipeline visibility.
- Show source coverage growth.
- Show collateral production velocity.
- Show meeting, outreach, proposal, application, and partner progress.
- Show traction by product line.
- Show governed approval and claims review.
