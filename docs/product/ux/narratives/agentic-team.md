# Venture OS Agentic Team

The agentic team is the product's service engine. Agents do preparation,
monitoring, drafting, scoring, and coordination. Humans retain authority over
external actions and strategic commitments.

## Team Roster

| Agent | Mission | Primary Outputs |
| ----- | ------- | --------------- |
| Venture Chief of Staff | Run cadence, priorities, briefs, approvals, blockers | Daily brief, weekly founder review, escalation list |
| Source Intelligence Agent | Monitor Africa/Global South funding, programs, events, partners, buyers, press | Signals, source freshness, opportunity candidates |
| Opportunity Analyst Agent | Score and route signals | Fit score, routing note, next action |
| CRM Operations Agent | Maintain client pipeline in ClickUp/app | Clean tasks, statuses, owners, stale-task report |
| Inbox + Calendar Agent | Summarize threads, create prep/follow-up tasks | Thread summaries, meeting prep, deadlines |
| Capital Desk Agent | Handle grants, VC, angels, DFIs, accelerators, challenges | Investor/application briefs, requirements checklist |
| Revenue Desk Agent | Prepare buyer, enterprise, channel, paid-pilot opportunities | Buyer brief, pilot proposal, commercial ask |
| Partnership Desk Agent | Prepare sovereign, DFI, bank, telco, university, corporate partnerships | Partner map, concept note, stakeholder brief |
| Visibility Desk Agent | Prepare LinkedIn, press, awards, speaking, newsletters | Content drafts, press pitches, event applications |
| Investor Relations Agent | Manage investor segmentation, warm intros, updates, diligence | Investor briefs, intro drafts, update drafts |
| Collateral Factory Agent | Create decks, one-pagers, proposals, concept notes, media kits | Draft assets, asset inventory, version history |
| Finance + Deal Economics Agent | Review budgets, pricing, valuation, terms, runway, revenue share | Economics memo, pricing options, term analysis |
| Claims + Proof Reviewer | Review claims, metrics, compliance, market assertions | Red flags, evidence links, approval readiness |
| Client Success Agent | Onboard clients, track maturity, maintain weekly outcomes | Client profile, maturity score, success report |

## Authority Levels

| Level | Agent Can |
| ----- | --------- |
| A0 Observe | Read assigned data and summarize |
| A1 Draft | Draft internal notes, collateral, and suggested actions |
| A2 Recommend | Score, prioritize, and recommend next actions |
| A3 Prepare | Create tasks, organize docs, prepare drafts for approval |
| A4 Execute Internally | Update CRM, labels, folders, statuses, and internal workflows |
| A5 External Action | Reserved for humans unless explicitly delegated |

## Human Approval Required

- External email, LinkedIn, WhatsApp, press, or partner sends.
- Applications and program submissions.
- Investor updates.
- Public collateral.
- Financial claims, valuation, pricing, terms, dilution, revenue share.
- Legal terms, exclusivity, commitments, MoUs, contracts.
- Regulatory, compliance, ESG, traceability, impact, or market-size claims.

## Handoff Contract

Every handoff must include:

1. Client.
2. Opportunity.
3. Requested output.
4. Audience.
5. Source links.
6. Deadline.
7. Approval needed.
8. Known risks.
9. Next action after completion.

## Client Operating Cadence

| Cadence | Output |
| ------- | ------ |
| Daily | Client command brief |
| Weekly | Pipeline review and next-action plan |
| Biweekly | Collateral and visibility review |
| Monthly | Investor update and growth metrics pack |
| Quarterly | Strategy refresh and opportunity-source recalibration |
