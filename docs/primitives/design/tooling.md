# Tooling

Tool radar for design, asset, motion, and agent-assisted creation.

## Direction

Record Venture OS-specific design decisions for venture operating system workflows, portfolio strategy, investor materials, and product operating cadence. Keep direction practical, reviewable, and grounded in real operator workflows.

## Required Notes

- State what is normative now versus exploratory.
- Link reference material when it changes design direction.
- Avoid decorative patterns that obscure workflow clarity, evidence review, or operational trust.
- Keep implementation details in owner product surfaces unless they define a reusable rule.
