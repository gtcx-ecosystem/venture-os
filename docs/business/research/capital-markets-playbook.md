# Capital Markets Playbook

## Positioning

Ecosystem OS should be presented as an AI-native company operating system with a defensible operating graph and evidence ledger.

## Investor narrative

1. AI agents are becoming enterprise actors.
2. Existing companies do not have operating infrastructure for agentic work.
3. Ecosystem OS provides the substrate: Kernel, Graph, Ledger, Surface.
4. Operating modules expand the platform: Ethos, Venture, Agency, Bridge, Agile, Fabric, Assurance.
5. Canon publishes proof and official knowledge surfaces.

## Proof requirements

1. Internal reference deployment.
2. Graph schema and example populated graph.
3. Ledger evidence envelope and replay examples.
4. MCP tool map.
5. Surface demo.
6. 11PR architecture audit.
