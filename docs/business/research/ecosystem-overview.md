# Ecosystem Overview

## Role in the ecosystem

Ecosystem OS is the operating layer for the AI-native company.

```text
Kernel   governs
Graph    knows
Ledger   proves
Surface  exposes

Ethos      judges
Venture    invents
Agency     expresses
Bridge     coordinates
Agile      ships
Fabric     operates
Assurance  verifies
Canon      publishes
```

The ecosystem is strongest when Canon, GTCX OS, Baseline OS, and Ecosystem OS have distinct roles:

| System       | Role                               |
| ------------ | ---------------------------------- |
| Canon OS     | Documentation and publishing OS    |
| Ecosystem OS | Company and ecosystem operating OS |
| GTCX OS      | Trade platform implementation      |
| Baseline OS  | AI/runtime substrate               |
